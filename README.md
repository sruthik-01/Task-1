API DATA DASHBOARD USING JAVA

1. OBJECTIVE

The objective of this project is to fetch age prediction data from a public API and display it to the user using a Java application.

2. SOFTWARE REQUIREMENTS

• Java JDK

• IntelliJ IDEA

• Internet Connection

3. PROJECT DESCRIPTION

This project uses a public API called Agify API. The user enters a name, and the application sends a request to the API. The API predicts the age associated with the entered name and returns the information in JSON format. The application then displays the received data to the user.

4. WORKING

Step 1: User enters a name.

Step 2: Java application creates the Agify API URL using the entered name.

Step 3: The application sends an HTTP GET request to the Agify API.

Step 4: The API processes the request and returns age prediction data in JSON format.

Step 5: The application reads the API response.

Step 6: The response is displayed on the screen.

5. OUTPUT

Example Input:

Name: Sruthi

Example Output:

API Response:

{"count":123,"name":"sruthi","age":25}

6. ADVANTAGES

• Easy to use

• Real-time data retrieval

• Simple implementation

• Helps understand API integration

• Improves knowledge of HTTP requests and responses

7. CONCLUSION

The API Data Dashboard project was successfully developed using Java and the Agify API. The project demonstrates how Java applications can communicate with web APIs, retrieve data dynamically, and display the received information to the user.
